## Best Paper Award
This research paper titled **“DHISASYA – An Intelligent Crop Prediction System”** received the **Best Paper Award** at the **7th International Conference on Recent Trends in Technology, Engineering and Applied Science (ICRTTEAS 2025)** held on **16–17 May 2025**.

## Project Description
Dhisasya is an AI-powered intelligent crop prediction system designed to help farmers make data-driven agricultural decisions and improve crop productivity.

## Technologies Used
- Python
- Machine Learning
- LSTM
- RNN
- CNN
- GAN

## Dataset
Kolar District agricultural dataset used for crop prediction.
